# DadBod

A progressive web app for tracking a weight loss journey as a busy dad.

## Features
- Onboarding with personalised plan calculations
- Weekly workout schedule (Mon/Wed/Fri mornings + Saturday)
- Daily streak tracker with motivational quotes
- Morning workout reminders + missed workout notifications
- Configurable water reminders
- Weight progress tracking with milestones
- Meal plan guide with dynamic calorie targets
- Backup & restore via export code
- Installable as home screen app (PWA)

## Stack
React 18 · Service Worker · localStorage · Netlify

## Dev
```bash
npm install
npm start
```

## Deploy
Connects to Netlify via this repo. Push to `main` auto-deploys.
